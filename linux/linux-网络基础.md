# 网络基础

## 网络接口层

| 项 | 说明 |
| --- | --- |
| 地址 | MAC 地址（48 位） |
| 基本数据单位 | 帧 |
| 主要协议 | 以太网协议 |
| 主要设备 | 交换机 |
| MTU | 以太网常见 1500 |

## 网络层

| 协议 | 作用 |
| --- | --- |
| IP | 网络层寻址与转发 |
| ARP | IP 解析为 MAC |
| RARP | 逆地址解析 |
| ICMP | 网络控制与差错报告（ping） |

主要设备：路由器。

## IP 地址分类

| 类别 | 地址范围 |
| --- | --- |
| A | `0.0.0.0` - `127.255.255.255` |
| B | `128.0.0.0` - `191.255.255.255` |
| C | `192.0.0.0` - `223.255.255.255` |
| D | `224.0.0.0` - `239.255.255.255`（组播） |

特殊地址：`255.255.255.255`（受限广播）、`127.0.0.1`（本机环回）。

## 私有地址

| 类别 | CIDR | 范围 |
| --- | --- | --- |
| A | `10.0.0.0/8` | `10.0.0.0` - `10.255.255.255` |
| B | `172.16.0.0/12` | `172.16.0.0` - `172.31.255.255` |
| C | `192.168.0.0/16` | `192.168.0.0` - `192.168.255.255` |

## 子网掩码与网络号

```text
IP 地址二进制 AND 子网掩码二进制 = 网络号
```

示例：`192.168.31.22 /24`

```yaml
网络号: 192.168.31.0
广播地址: 192.168.31.255
主机范围: 192.168.31.1 - 192.168.31.254
```

### 子网计算示例

`192.168.31.46 / 255.255.255.248`（块大小 8，46 落在 40-47）：

```yaml
网络号: 192.168.31.40
可用主机: 192.168.31.41 - 192.168.31.46
广播地址: 192.168.31.47
```

`10.100.111.245 / 255.255.192.0`（第三段块大小 64，111 落在 64-127）：

```yaml
网络号: 10.100.64.0
可用主机: 10.100.64.1 - 10.100.127.254
广播地址: 10.100.127.255
```

## 路由表

```bash
route -n
```

## 临时静态路由

```bash
route add -net 22.22.22.0 netmask 255.255.255.0 gw 192.168.8.1
```

## 永久静态路由

```bash
/etc/sysconfig/network-scripts/route-eth0
```

```text
10.15.150.0/24 via 192.168.8.1 dev eth0
```

## 虚拟 IP

```bash
ifconfig ens33:1 192.168.8.5 netmask 255.255.255.0 up
```

临时生效，重启丢失。长期配置应写入网卡配置文件。

## NAT

NAT 用于内网私有地址访问公网、多台主机共享公网出口、端口映射。

## 易错点

| 场景 | 说明 |
| --- | --- |
| 子网掩码 `/25` 写成 `255.255.255.0` | `/25` 应是 `255.255.255.128` |
| 网络号把主机位算进去 | IP 与子网掩码按位与后才是网络号 |
| 临时路由/虚拟 IP 重启丢失 | 永久配置要写配置文件 |
